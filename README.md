# techbook_03_eventloop_code

「カチャカチャしながら Web ブラウザと JavaScript を体系的に学ぶ ② 後編」のサンプルソースコードです。

## 使い方

1. このリポジトリをクローンまたはダウンロードします
2. 各章のフォルダにある HTML ファイルをブラウザで開きます
3. `fetch` を使うコード例はローカルサーバーが必要です:
   - VS Code の Live Server 拡張を使う（最も手軽）
   - ターミナルで `npx serve` を実行する

## フォルダ構成

```
chapter1/   第1章 この後編で使う地図をそろえる
chapter2/   第2章 なぜ「書いた順」と「動いた順」がちがうのか
chapter3/   第3章 関数を呼んだ瞬間、エンジン内部に何が積まれるか
chapter4/   第4章 「終わったあと」を値として扱う
chapter5/   第5章 どの変数が「見える」か、どこまで生き残るか
appendix-a/ 付録A シューティングゲーム Lite
sample.gif  fetch のサンプルで使う画像ファイル
```

## 動作確認環境

- ブラウザ: Chrome 142 以降
- 端末: MacOS Tahoe 26.1 で動作確認済み
